# 数学错题整理本
## 文档基础说明
- 适用学段：初中数学
- 收录范围：单元测试、同步练习、中考模拟、课堂作业错题
- 统一记录字段规范：
  1. 整理日期：记录抄写错题当天日期
  2. 题目出处：试卷/练习册名称+题号
  3. 题型分类：标注对应题型模型
  4. 自我作答答案：自己当初写错的答案
  5. 标准参考答案：题目正确答案
  6. 解析二维码：配套讲解二维码，无则填写「无」
  7. 原题题干：完整题目原文
  8. 错误原因：细分错因，写明思维漏洞
  9. 标准解析：完整解题步骤
  10. 复盘总结：考点、避坑技巧、同类题思路

## 符号约定
1. 📷：代表此处附有解析二维码图片
2. ⚠️：高频易错点标记，同类题重点留意
3. 🔴：典型易混模型，需要反复回看
4. ⭐：此处是前面错点
